VehicleReader
=============

A simple web scraper that read the references, brand, model and year/price of vehicles from http://www.fipe.org.br/

### Deploy structure

```
VehicleReader/
├── VehicleReader.jar
├── config.ini
```

### config.ini structure

- connection.dbms `MariaDB` or `SQLServer` (optional)
- connection.host
- connection.database
- connection.user
- connection.password
